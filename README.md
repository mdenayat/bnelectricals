

# BN Electricals - Company Portfolio Website

This is a modern and responsive company portfolio website developed for **BN Electricals**.  
The project showcases company details, services, projects, testimonials, and other essential sections that highlight the brand’s profile.  

The website is built using **React.js**, styled with **Tailwind CSS**, and optimized with **Vite** for fast builds and performance. It is hosted on **Vercel** for seamless deployment and scalability.

---

## Features

- Responsive design for all devices
- Company overview and statistics
- Services and project showcase
- Contact form
- Testimonials section
- Interactive gallery with lightbox
- Smooth navigation and user-friendly interface

## Installation and Setup

Follow these steps to set up the project locally:

1 Clone the repository
- git clone https://github.com/mrsaffu/BN_ELECTRICALS_WEB.git

2 Navigate to the project folder
- cd bn-electricals

3 Install dependencies
- npm install

4 Start the development server
- npm run dev

5 Open the app
- Visit http://localhost:5173 in your browser.
